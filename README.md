# Autodrills Preview

Autodrills Preview is a generated public preview of the Autodrills practice experience.

This site contains a curated subset of the full Autodrills corpus so teachers and maintainers can inspect the public-facing experience without exposing the complete private build pipeline.

## What this preview includes

- A small curated drill catalog
- Preview practice session flow
- Drill reporting through GitHub Issues
- Public-facing screenshots and upgrade context

## What this preview does not include

- Student identity
- Server-side persistence for responses or reports
- The full private corpus
- Private repo-only data, secrets, or infrastructure

## Important notes

- The full Autodrills product remains free, but it requires setup in the private repository and local workspace.
- Canonical development happens in the private Autodrills repository, not here.
- Drill reports open GitHub Issues in the public preview repo instead of posting to a private service.

## Working with the preview

This repository is generated output. If the private preview build changes, regenerate and resync the public site from the private repo.

For maintainers, the source of truth remains the private Autodrills workspace.
